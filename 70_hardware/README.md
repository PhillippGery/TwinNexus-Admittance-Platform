# 70_hardware

*Mechanical and physical-integration material for the bimanual robot cell.*

## Contents

- `cad/` - camera mount geometry, adapter plates, and printable or machinable components
- `bom/` - bill of materials and part sourcing references
- `lab_layout/` - workstation placement, cell footprint, and physical constraint references

## Conventions

- CAD assets should include revision-aware filenames and exported interchange formats where practical.
- BOM files should distinguish custom, printed, off-the-shelf, and fastener components.
- Physical-layout material should record measurement units and coordinate assumptions explicitly.
- Hardware notes should remain tied to actual lab constraints rather than generic robotics templates.
